# Programmation Web

Ce dépôt contient les travaux pratiques (TP) réalisés dans le cadre du module **Programmation Web**.

## 📚 Contenu du projet

Le repository est organisé en plusieurs dossiers correspondant aux différents TP :

- **tp2 html** : exercices et exemples en HTML
- **tp3 css** : mise en forme et styles avec CSS
- **tp4 js1** : introduction à JavaScript
- **tp4 js2** : manipulation avancée avec JavaScript
- **tp5 http** : notions sur le protocole HTTP
- **tp6 php1** : introduction à PHP
- **tp6 php2** : programmation PHP avancée
- **tp7 bd** : interaction avec la base de données

## 🛠 Technologies utilisées

- HTML
- CSS
- JavaScript
- PHP
- Base de données

## 🎯 Objectif

L'objectif de ce projet est de pratiquer les concepts fondamentaux du **développement web**, notamment :

- la création de pages web
- le design avec CSS
- la programmation côté client avec JavaScript
- la programmation côté serveur avec PHP
- la communication avec une base de données
