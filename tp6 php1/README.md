# 🐘 TP6 – PHP : Premiers pas avec le langage

## 🎯 Objectif du TP

L'objectif de ce TP est de vous familiariser avec les **bases du langage PHP**.  
Vous allez mettre en pratique ces notions à travers **2 exercices progressifs**.

---

## 📘 Ressources fournies

- Le fichier **`TP6_PHP1.pdf`** vous guidera **étape par étape** dans la réalisation du TP.
- Tous les **scripts PHP** doivent être écrits dans un fichier nommé **`index.php`**.
- Les **noms des id, classes et variables** doivent correspondre **minutieusement** à ceux donnés dans chaque exercice.
- Votre code doit impérativement être **commenté**, en mentionnant pour chaque concept utilisé le **numéro de la diapositive correspondante** du cours.

> Exemple :  
> `// Ici on utilise une boucle (vu dans le slide 45)`

🛑 **Un code sans commentaire est automatiquement rejeté.**

---

## 🧭 Déroulement du TP

1. **Accès à l'énoncé**
   - L'énoncé complet du TP est déposé sur **GitHub**.
   - Le lien de l'assignement est communiqué sur **Google Classroom**.

2. **Création de votre espace de travail**
   - Acceptez l'assignement sur GitHub Classroom.
   - Clonez ensuite votre dépôt individuel.

3. **Réalisation**
   - Complétez le fichier **`index.php`** dans **chaque dossier d'exercice**.
   - Suivez les consignes décrites dans **`TP6_PHP1.pdf`**.

4. **Soumission**
   - Une fois le travail terminé :
     ```bash
     git add .
     git commit -m "TP6 PHP terminé"
     git push origin main
     ```

---

## 🧾 Livrables attendus

- Un **dossier par exercice** (`exo1`, `exo2`).
- Dans chaque dossier doit se trouver : **`index.php`**

## 🧠 Rappel

Avant de soumettre :

- Vérifiez que **votre code s'exécute sans erreur** dans le navigateur.
- Respectez les **noms d'id, classes et variables imposés**, l'**indentation** et les **commentaires** exigés.
- Chaque script doit être **autonome** et respecter les **bonnes pratiques vues en cours**.
