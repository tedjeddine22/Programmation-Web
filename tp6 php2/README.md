# ⚙️ TP6 – PHP (part 2) : Manipulation des formulaires côté serveur

## 🎯 Objectif du TP
L’objectif de ce TP est de **manipuler les formulaires en PHP**.  
Vous allez mettre en pratique ces notions en **reconstruisant l’application CRUD** déjà développée lors du **TP4_2 en JavaScript**, mais cette fois **entièrement côté serveur avec PHP**.

---

## 📘 Ressources fournies
- Le fichier **`TP6_PHP2.pdf`** vous guidera *étape par étape* dans la réalisation du TP.
- Tous les fichiers PHP doivent être soigneusement commentés en indiquant, pour chaque concept utilisé, le **numéro de slide correspondant du cours**.  
  > Exemple :  
  > `// Ici on utilise une variable superglobale (vu dans le slide 12)`

🛑 **Un code sans commentaires est automatiquement rejeté.**  
🛑 **Les noms des variables, IDs, classes et champs doivent correspondre EXACTEMENT à ceux demandés dans l’énoncé.**

### 1. Accès à l’énoncé
- L’énoncé détaillé est disponible sur **GitHub**.

### 2. Objectif de réalisation
- Reproduire le mini-projet CRUD déjà développé en JavaScript (**TP4_2**).

- Respect strict de la **structure des fichiers** demandée.
- Respect strict des **noms de champs**, **IDs**, **classes CSS** et **variables PHP**.
- Validation des données **côté serveur obligatoire**.
